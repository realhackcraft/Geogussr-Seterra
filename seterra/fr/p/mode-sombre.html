Nouvelle fonctionnalité : le mode sombre !

À l’origine, le site Seterra a été conçu en couleurs claires, mais certains pensent que les couleurs plus sombres sont plus agréables à l’œil, ou plus confortables à regarder. Désormais vous pouvez choisir entre le mode sombre ou le mode clair sur le site Seterra !


Pour activer le mode sombre, cliquez sur la case à cocher dans le coin supérieur droit. Si vous visitez le site à l’aide d’un navigateur sur votre téléphone, vous devrez au lieu de cela chercher le bouton du mode sombre dans le menu déroulant.

Le mode sombre est par ailleurs légèrement plus compact que le mode clair, ce qui nécessite moins de défilement à l’écran.
