Les modes de jeu du site Seterra


Regardez la vidéo de démo de sept parmi les neuf modes de jeu



Neuf modes de jeu sur le site Seterra vous défient de différentes façons pour vous aider à retenir l’information et à rendre les choses intéressantes.

Montrer tout
Affichez tous les noms de lieux sur la carte.

Découvrir et acquérir
Étudiez les États ou les pays sur une carte grâce au mode “Découvrir et acquérir”, qui affiche le nom d’un pays ou d’un État pendant un court laps de temps après avoir cliqué dessus.

Choix multiples
Il s’agit d’un mode assez facile qui peut être utile pour apprendre une nouvelle carte. Un emplacement sur la carte est mis en évidence en violet, et vous avez le choix entre quatre options. Cliquez sur le bon bouton ou appuyez sur les chiffres 1 à 4 du clavier pour répondre à la question. Il s’agit d’un nouveau mode - il a été ajouté en novembre 2020.

Repérer
Ce mode de jeu chronométré vous met au défi de trouver sur la carte de votre choix les États ou les pays en cliquant sur leurs emplacements lorsque vous pensez le connaître. Les scores s’affichent à la fin de chaque tour et indiquent le pourcentage de bonnes réponses.

Code couleur :

   Correct en 1 clic

   Correct en 2 clics

   Correct en 3 clics

    > 3 clics nécessaires


Repérer (difficile)
Notre mode le plus difficile ! Ce mode de jeu est similaire au mode “Repérer”, mais après avoir donné une réponse correcte, l’emplacement reprend sa couleur verte d’origine, de sorte que vous ne pouvez pas utiliser l’élimination lorsque vous jouez les derniers emplacements.

Étiqueter
Placez les étiquettes pour les pays ou les États un par un dans l’ordre de votre choix dans le mode de jeu “Étiqueter”. Voyez la liste diminuer au fur et à mesure que vous parvenez à étiqueter chaque contour.

Écrire (facile)
Tapez les noms des lieux dans n’importe quel ordre et ils apparaîtront automatiquement au bon endroit sur la carte ! Il s’agit d’un nouveau mode - il a été ajouté en avril 2020.

Écrire (avec autocomplétion)
Tapez quelques lettres du nom du pays sur votre clavier et vous obtiendrez une liste d’options parmi lesquelles choisir. C’est parfait si vous aimez taper au clavier, mais que vous trouvez fastidieux de taper chaque lettre pour "les États fédérés de Micronésie" :) Il s’agit d’un nouveau mode - il a été ajouté en novembre 2020.

Écrire
Dans le mode “Écrire”, vous tapez les noms des lieux sur votre clavier. Vous ne vous souvenez pas d’un nom ? Cliquez sur le bouton “Indice” pour obtenir un indice !

Astuce ! La boîte recouvre-t-elle un élément intéressant de la carte lorsque vous utilisez le mode “Écrire” ? Utilisez les touches fléchées pour la déplacer vers le haut et le bas, ou utilisez les touches Ctrl+flèche pour la déplacer latéralement.

Dans les versions iOS et Android, les modes “Découvrir et acquérir” et “Repérer” sont disponibles. Dans la version Windows, seul le mode “Repérer” est disponible.

Changez le mode de jeu en utilisant la liste déroulante dans le coin inférieur gauche.
