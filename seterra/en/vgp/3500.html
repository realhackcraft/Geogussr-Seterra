Can you identify all 67 counties in Alabama? This quiz game will make it easy and learning a few state facts will help too. Limestone County, Alabama is home to a state welcome center featuring a genuine Saturn 1B Rocket. The Boll Weevil Monument, located in Dale County, is the first monument erected in honor of an agricultural pest. In Mobile County, you'll find a well-preserved, decommissioned battleship, the USS Alabama. There are interesting landmarks in the state known as "The Heart of Dixie"—if you commit a few details to memory and work with this map quiz, you'll be able to identify all 67 counties of Alabama in no time.
